Machine Learning

常见知识

  • 不等于 ~= ，不是 !=

  • 使用ps1("xxx")=====>改变提示方式

  • 加上分号(；)可以阻止打印

  • 生成一个从1到2，步长为0.1的 行向量

  • rand（row，col）和randn（row，col）区别：后者为正态分布（高斯分布）

  • hist（）命令 ：生成一个直方图，可以看出，满足高斯分布

  • eye（n）：生成n阶单位矩阵

  • help 加上函数名 ： 可以获得相关函数的帮助文档

  • size（矩阵名）：返回一个行和列的矩阵表示--->即1 * 2 的矩阵

  • length（xxx）：返回矩阵中较大的那个维度，一般对向量才使用

  • pwd ：显示当前路径

  • cd 命令：修改路径

  • ls：显示当前路径的列表

  • 加载文件的两种方式 ：load 的不同使用

  • who ：显示当前在内存中存储的变量（包括文件名）

  • whos：显示详细信息

  • clear的用法

    • clear xxx：删除某个变量
    • clear ： 删除全域

  • v = x（1:10）--->将x的前十个元素赋值给v

  • save 目标文件名 变量(即待存储的值)：将某个元素存入目标文件

  • 以ascii格式存储文件

  • A (3,:) ：获得A 的第三行的所有元素，‘：’代表某行或者某列的所有元素

  • A (3,:) ：选取特定的行/列，使用 [ ]表示

  • A（ : ）：将A 的所有元素放入一个列向量中

  • 合并两个矩阵

    • C = [A B]:水平合并
    • C= [A ; B]:竖直合并

  • A .* B ：将A和B中的对应元素相乘，不是矩阵乘法，注意 . 的加入（针对元素的操作）

  • % ：代表注释

  • C ' :求C 的转置

  • 使用val表示max值，ind为其对应的索引（例子中的a为列向量，若为多列矩阵，返回每一列的最大值）

  • a < 3 ：返回矩阵对应元素进行 <3 的判断的逻辑结果（0和1）

  • find（xxx）,找到某个满足xxx表达式的索引，可用[r, c]来返回行和列的索引

  • sum( a)求和, prod(a )求乘积，

  • floor()向上取整 ,ceil()向下取整

  • 求每一行/列的最值

  • 因为对于一个多维矩阵来说，调用max返回的是各列对应的max值构成的矩阵，所以要想获得全局的max值，使用max（max（A））即可得到

  • 一般在参数中-> 1代表列,2代表行 e.g. sum（A ,1）:计算列的和

  • flipud（）：对矩阵进行垂直翻转

    • e.g.求A 矩阵副对角线的和：
  • pinv（A ）: 求A 的伪逆矩阵

画图

  • 使用plot（）进行画，==》plot（t，y1），参数分别代表x轴和y轴

  • hold on的用法，以及指定图示的颜色

  • xlabel，ylabel ：xy轴的标签

  • legend（' ',' '）:添加图形的注解

  • title（）：添加标题
  • 保留图片，可以修改路径
  • close ：关闭图像的命令
  • 生成两个图像，指定figure的参数
  • 使用子图的形式：subplot（1,2,1）：签名的1 ，2 代表 生成一个1*2的格子，后面的1代表我现在使用的是第一个格子。使用第二个格子：subplot（1,2,2）

  • 指定xy轴刻度范围

  • clf ： 清除一个图像

  • 使用图像来表示矩阵

    • e.g. 灰度图

  • 连续执行三个命令：使用，分隔开即可

控制语句

  • for：i = 1:10，end; (代码的空格没有影响)

  • 同样的效果，使用indices代替1:10

  • while & if ： 记得if也要使用end语句

  • disp( ):输出语句
